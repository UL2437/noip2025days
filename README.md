### 总结

今天暴露四个问题：

1. **贪心题有思路额外看下发样例** : 因为贪心毕竟不稳定，一定要确保把自己的核验资源用干净了，证伪不了自己的贪心再写！

3. **灵活切换思路和想法** : 像 $Whk$ 的策略一样，一切以拿分为目标！

4. **万不得已时记得多元发展** : ~~以至于乱搞~~
